VIHIS is an AI-powered mobile application designed to support Vietnamese history learning through personalized content, real-time interaction, and intelligent knowledge assistance.

###  Project Overview
The project focuses on analyzing business requirements and translating them into a scalable mobile learning system. VIHIS combines quiz-based learning, real-time PvP competition, and AI-powered knowledge support to enhance user engagement and learning effectiveness.

###  Technologies & Tools
- **Mobile Development:** Kotlin, Android Studio  
- **Backend & API:** Flask API  
- **Database & Cloud:** Firebase Firestore, Firebase Realtime Database, Firebase Auth, FCM  
- **AI Integration:** Ollama (Gemma 2, LLaVA)  

###  Key Features
- Personalized quiz-based learning experience  
- Real-time PvP arena with scoring, ranking, and room-based competitions  
- Firebase-based data models for users, quizzes, learning progress, and real-time battles  
- Multilingual AI chatbot (text & image) for knowledge lookup and contextual Q&A  
- End-to-end data flow between Android app, Firebase services, backend API, and AI components  

###  Role & Responsibilities
- Analyzed and documented business requirements  
- Designed system architecture, workflows, and database models  
- Developed core mobile features and integrated AI chatbot  
- Collaborated on API design and data flow definition across system components  

###  Results
- Delivered a scalable AI-powered mobile learning platform  
- Increased user engagement through personalization and real-time interaction  
- Improved learning experience with intelligent, context-aware AI assistance
